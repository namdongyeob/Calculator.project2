#  Java 계산기 프로젝트 (Calculator)

입력받은 양의 정수와 사칙연산 기호를 통해 계산을 수행하고 결과를 저장/관리하는 계산기 입니다.

---

##  주요 기능

### 1. 사칙연산 수행
`+`, `-`, `*`, `/` 기호를 입력받아 연산을 수행합니다.

예외 처리: 0으로 나누기에서 프로그램이 종료되지 않도록 `try-catch`를 사용하여 에러 메시지를 출력합니다.

### 2. 양의 정수 입력 검증
입력값이 0보다 작을 경우, `continue`를 활용해 사용자에게 다시 입력을 요청합니다.

### 3. 결과 기록 관리 (ArrayList)
* 계산 결과를 리스트에 순서대로 저장합니다.
* `조회` , `삭제` , `초기화`
### 4. 반복 및 종료
* 사용자가 `exit`를 입력할 때까지 계산을 반복하며, `while`와 `break`를 통해 멈추거나 계속하기를 실행합니다.

---


## 📂 프로젝트 구조
- `App.java`: 사용자의 Main 클래스
- `Calculator.java`: Main 클래스의 기능을 담당하는 클래스
